# PRWT

Phrase Ranker With Template

## Requirements

- Node.js 25
- npm 11
- Angular CLI 20
- Docker

## Run locally (Angular dev server)

```sh
cd client
npm run start
```

Open http://localhost:4200

## Run with Docker

```sh
docker compose up
```

Open http://localhost:8080
